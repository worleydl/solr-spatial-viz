About this Project

This app is a simple yet effective way of demonstrating the power of pairing Solr with a Map visualization. Utilizing data from HMDB (The database of Historical Markers) and OpenLayers (A powerful opensource mapping tool), it's possible to provide users with interesting visualizations of events in US history.

The code behind this application is intended to be easy to follow. Essentially all that's happening behind the scenes is a basic query to Solr, converting the response to GeoJSON then feeding it into OpenLayers.
